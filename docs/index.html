How it works

Install once. Run anywhere.

PositioningHQ is a standard Claude Code plugin — no APIs, no accounts, no configuration.

1

Install the plugin

Run /install-plugin github.com/your-github-handle/positioninghq in any Claude Code session. Done.

2

Set up your vault

Run /setup-positioning-vault — it creates the folder structure, asks for company details, and auto-runs research + playbook in one session.

3

Generate your playbook

Run /positioning-playbook. The skill auto-loads your company research and asks for any supporting materials (pitch deck, case studies, FAQs).

4

Equip your sales team

Run /battlecard and /call-plan. Both skills auto-load your playbook from memory — no copy-pasting required.
